# OK Content Manifest

State: Oklahoma
Tier: 4 (Blocked — missing state-specific content)
Requirement: 24 CE hours per 2-year license term
- Ethics: 3 hours minimum (mandatory)
- Legislative update: 2 hours (mandatory each cycle, state-specific)
- Total mandatory specialty hours: 5 hrs (3 ethics + 2 legislative update)
- Self-study split: No formal split requirement
- Proctored exam: Not required
- Carryover: Up to 6 hrs to next renewal period
- CE due: 30 days before license expiration

Specialty requirements:
- LTC: 8-hr initial + 4-hr ongoing
- Annuity: 4 hrs (1-time)

Launch status: BLOCKED pending OK Legislative Updates course creation (2 hrs, mandatory each cycle).
The generic ETH course satisfies the 3-hr ethics requirement. 19.17 hrs of additional generic content is available.
See MISSING CONTENT section at the end.

## ⚠️ MISSING CONTENT — BLOCKED

### What is missing:

**OK.INS-CE-LEG — Oklahoma Legislative Updates (2 hrs)**
- Mandatory each renewal cycle for ALL OK-licensed producers
- Must cover: Recent Oklahoma insurance legislation, OK DOI regulatory updates, new statutes and rules from the current biennium
- Content changes each renewal cycle (must be updated with new legislation)
- State-specific — cannot be substituted with generic material
- This is a HARD BLOCK: OK program cannot launch without this course

### Additional specialty courses needed (not launch blockers):

**Annuity CE (4 hrs)** — One-time requirement. Separate course needed.

**LTC (8-hr initial + 4-hr ongoing)** — For LTC-licensed producers. LHOPT L4 provides partial credit (2.16 hrs). Dedicated LTC supplement courses needed.

### What this unblocks:

Once OK.INS-CE-LEG is built and approved by OK DOI:
- Full OK program can launch
- OK ETH (2.70 hrs generic) satisfies the 3-hr ethics requirement — no additional ethics content needed
- OK.INS-CE-LEG (2 hrs) satisfies the legislative update requirement
- Total with both: 23.87 + 2.00 = 25.87 hrs (above 24-hr minimum)
- Carryover: Up to 6 hrs carry to next renewal period
- CE due 30 days before expiration — ensure learner communications reflect this
- OK.INS-CE-LEG will need to be updated each renewal cycle with current legislative changes
