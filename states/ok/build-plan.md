## What Needs to Be Built

### OK.INS-CE-LEG — Oklahoma Legislative Updates
**2 hrs | Mandatory each renewal cycle | Self-study or classroom equiv**

This is the only blocker. It must cover recent Oklahoma insurance legislation, OK DOI regulatory updates, and any new statutes or rules from the current biennium.

**Content scope (2 hrs = ~18,000 words at Basic formula):**
- Changes to Oklahoma insurance statutes (Title 36, OK Statutes)
- New OK DOI rules, bulletins, and regulatory guidance issued this cycle
- NAIC model law adoptions by Oklahoma (if any)
- Consumer protection updates and new compliance requirements
- Significant OK court decisions or enforcement actions affecting producers

**Important:** This course must be **updated every 2 years** at renewal cycle. It cannot be a static course — the content is tied to the current legislative biennium. Build a process for updating it each cycle.

**Delivery:** Self-study is simplest (add final exam, no timing/progression required). Could be classroom equivalent if preferred — would require checkpoints.

**Questions needed (self-study):**
- Minimum 20 questions (10 per credit hour × 2 hrs)
- 70% passing score
- No proctored exam required in OK

**Build steps:**
1. Research current OK insurance legislation (Title 36, OK Statutes — oklegislature.gov)
2. Pull OK DOI bulletins and rulemaking notices from this biennium (ok.gov/oid)
3. Write 2-hr course (~18,000 words) covering the scope above
4. Write 20 minimum exam questions
5. Submit to OK DOI via Sircon for approval

## OK-Specific Delivery Notes

- **No classroom/self-study split mandate** — full flexibility on delivery mix
- **CE due 30 days before license expiration** (earlier than TX's 2-week window)
- **No proctored exam required** — self-study exams are open-book, unmonitored
- **Up to 6 hrs carryover** to next renewal period
- **Annuity (1x 4-hr) and LTC (8-hr initial + 4-hr ongoing)** are one-time specialty requirements outside the base CE program — separate courses needed eventually, not launch blockers

## Ongoing Maintenance

**OK.INS-CE-LEG must be revised every 2 years.** Each new renewal cycle, the legislative update content becomes outdated. Build this into the product calendar — OK CE will need a content refresh every biennium. Plan for ~1 day of content work per cycle to update the 2-hr course with new legislation.
