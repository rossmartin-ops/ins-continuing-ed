## ⚠️ MISSING CONTENT — BLOCKED

### What is missing:

**MT.INS-CE-REG — Montana Insurance Statutes & Administrative Rules Update (1 hr)**
- Mandatory each renewal cycle for ALL MT-licensed producers
- Must cover: Montana insurance statutes, Montana Department of Commerce / State Auditor's Office insurance division regulations, recent changes to MT administrative rules
- State-specific — cannot be substituted with generic material
- Content should be updated each renewal cycle to reflect any statutory or regulatory changes
- This is a HARD BLOCK: MT program cannot launch without this course

### Additional specialty courses needed (not launch blockers):

**Annuity CE (4 hrs)** — One-time requirement. Separate course needed.

**LTC (8-hr initial + 4-hr ongoing)** — For LTC-licensed producers. LHOPT L4 provides partial credit (2.16 hrs). Dedicated LTC supplement courses needed.

### What this unblocks:

Once MT.INS-CE-REG is built and approved by MT Commissioner of Securities and Insurance:
- Full MT program can launch
- GEN.INS-CE-ETH (2.70 hrs) satisfies the 3-hr ethics requirement
- MT.INS-CE-REG (1 hr) satisfies the statutes/admin rules update requirement
- Total: 23.87 + 1.00 = 24.87 hrs (slightly over — trim generic content if needed, or accept 24.87)
- No carryover — all hours must be earned each cycle
- MT.INS-CE-REG will need to be reviewed and updated each renewal cycle
